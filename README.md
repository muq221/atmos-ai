# Atmos AI

## Overview
Atmos AI is a basic data engineering project that tracks how WEATHER data changes over time.

## Features
- Collects real-time data
- Stores past snapshots
- Calculates data drift
- Classifies change levels
- Stores results in Firebase cloud

## Tech Stack
- Python
- Pandas
- Firebase
- API

## Output
The system detects and classifies data changes and stores them in the cloud.

## Future Scope
- Real-time automation
- AI-based prediction
- Web dashboard
